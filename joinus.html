JOIN US

我们为每一位加入团队的队员提供独挡一面的发展机会。

首先我们会根据加入者自身特点，与每位深度沟通，确定发展目标。
我们会根据目标制定具体的分步实现计划，提供相应培训，提供每一步的发展平台，最终实现员工的职业发展计划。

  • 区域经理

    1. 1、 承担直管区域的年度经营目标拟定、分析工作；
    2. 2、 组织直管区域客户发掘，合同谈判和签定，业务的经营、促销的制定及管理工作；
    3. 3、 组织直管区域客户、市场网络管理，客户销售培训、顾问管理工作；
    4. 4、 其他总公司分派到各区域的项目性工作；

  • 财务经理

    1. 1、 组织预算、财务的执行与差异分析；
    2. 2、 督促和指导各区域客户管理；
    3. 3、 提供各区域经营决策的财务支持；
    4. 4、 维护公司资产的安全性和完整性；
    5. 5、 指导和监督财务档案管理、归档和保管等工作；

  • 产品经理

    1. 1、 沟通科技团队与生产商，承担产品开发、改进工作；
    2. 2、 制定产品使用推广计划，协调各部门整体实现产品运作；
    3. 3、 产品使用跟踪，制定产品使用标准流程；
    4. 4、 产品使用培训工作；

  • 文员

    1. 1、 接受客户咨询，并记录整理及汇报；
    2. 2、 协助区域经理做好客户资料收集与整理，订单管理，跟踪客户动态；
    3. 3、 熟悉公司办公流程，协助对接各外事部门；
    4. 4、 协助广告设计及出品
